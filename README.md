# dwm personalizada

Esta es mi build personalizada de [dwm](https://dwm.suckless.org/), el gestor de ventanas dinámico de suckless.

## Créditos

Esta configuración está **fuertemente basada** en el excelente trabajo de [BreadOnPenguins](https://github.com/BreadOnPenguins/dwm). Tomé su repositorio como base inicial para construir mi propia versión, adaptándola a mis necesidades y flujo de trabajo en Arch Linux y Ubuntu.

> Créditos y agradecimiento a su autor original por compartir su configuración pública.

## Requisitos

- libx11, libxft, libxinerama

- make, gcc

- Terminal compatible (uso recomendado: alacritty)

- dwmblocks para mostrar estado en la barra
